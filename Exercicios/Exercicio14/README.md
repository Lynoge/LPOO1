# Exercício 14 - Porta
Crie a seguinte classe:

Classe: Porta 
Atributos: aberta (boolean), cor (String)
Métodos:
* void abre()
* void fecha()
* void pinta(String s)
* boolean estaAberta()
* String getCor()
Para testar, crie uma porta, abra e feche a mesma, pinte-a de diversas cores e use o método estaAberta para verificar se ela está aberta.
